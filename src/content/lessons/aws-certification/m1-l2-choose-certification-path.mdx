## The Big Picture

You can earn up to **4 free vouchers total** — one per unique Official Practice Exam completed in Skill Builder. Each voucher covers a **100% discount** on a real AWS certification exam (a $150 value each, up to $600 total).

The eligible certifications fall into two levels:

---

## Foundational Level (Start Here If You're New)

These certifications have no prerequisites and are designed for anyone, regardless of technical background.

### AWS Certified Cloud Practitioner (CLF-C02)

- **Best for:** Complete beginners to cloud computing
- **What it covers:** Cloud concepts, AWS core services, pricing, support plans, and basic security
- **Exam duration:** 90 minutes · 65 questions
- **Why take it first:** It builds the vocabulary and mental model you need for every other AWS cert

### AWS Certified AI Practitioner (AIF-C01)

- **Best for:** Students interested in AI, machine learning, or generative AI on AWS
- **What it covers:** AI/ML fundamentals, generative AI concepts, AWS AI services (Bedrock, SageMaker, Rekognition)
- **Exam duration:** 120 minutes · 85 questions
- **Why take it:** Great pairing with any data science or AI coursework

---

## Associate Level (After You Have the Fundamentals)

These certifications are more technical and benefit from hands-on AWS experience.

### AWS Certified Solutions Architect – Associate (SAA-C03)

- **Best for:** Students interested in infrastructure, system design, or cloud architecture
- **What it covers:** Designing resilient, scalable systems on AWS — EC2, S3, RDS, VPC, IAM
- **Exam duration:** 130 minutes · 65 questions

### AWS Certified Developer – Associate (DVA-C02)

- **Best for:** Software developers building and deploying applications on AWS
- **What it covers:** Lambda, API Gateway, DynamoDB, CodePipeline, CloudFormation, IAM for developers
- **Exam duration:** 130 minutes · 65 questions

<Pitfall title="Specialty Exams Are Not Covered">

Certifications like **AWS Certified Security – Specialty** or **AWS Certified Machine Learning – Specialty** are **not** included in this voucher program. Only the four certifications listed above are eligible.

</Pitfall>

---

## How to Choose

| Your situation | Start with |
|---|---|
| New to cloud entirely | Cloud Practitioner (CLF-C02) |
| Interested in AI/ML or data | AI Practitioner (AIF-C01) |
| Want to design cloud systems | Solutions Architect – Associate |
| Building apps and APIs | Developer – Associate |
| Want all 4 vouchers | CLF → AIF → SAA → DVA (in that order) |
