## How to Use This Assessment

These three scenarios are deliberately hard. Each one is built the way a real internal-affairs review or a defense cross-examination would build it: a reasonable-sounding shortcut, taken under real operational pressure, that turns out to create exactly the exposure this course has spent five modules explaining.

Read each scenario fully before choosing. The wrong answers are not absurd — they're the choices an exhausted, well-intentioned officer or supervisor might actually make.

<Pitfall title="Before You Start">

None of these scenarios have a "just don't use AI at all" answer available as the safe default. That's intentional. The compliant answer in every scenario still involves AI — just used correctly, inside the AI-as-Auditor model from [Module 5](/operational-ai-compliance/lessons/m5-l1-the-ai-as-auditor-workflow). "Avoid the tool entirely" is not the lesson of this course; "know exactly which use is safe" is.

</Pitfall>
