## Key Glossary Terms

Below are the most important terms from this course. Review them before taking the final quiz.

| Term | Definition |
|------|-----------|
| **Abrasion (Hose)** | Wearing, grinding, or rubbing away of material from the hose cover |
| **Accumulator** | Pressurized storage container for fluids; absorbs surges and maintains constant pressure |
| **Ambient Temperature** | Temperature of the hydraulic system's surrounding environment |
| **Bend Radius** | Radius measured between the innermost surfaces of a curved hose; minimum bend radius is how far the hose can bend and still meet impulse test requirements |
| **Bend Restrictors** | PVC sleeves near the coupling to reduce stress at the hose/coupling interface and prevent kinking |
| **Bite Sleeve** | Internal sleeve used in flareless assemblies and standpipe couplings |
| **Block Style Fittings** | Use bar stock to orient the desired angle; increase pressure drop more than tube fittings |
| **BSPP** | British Standard Pipe Parallel — straight threads, 30° seat, O-ring or soft metal washer seal |
| **BSPT** | British Standard Pipe Tapered — tapered threads; seal made on threads; identical to Japanese tapered pipe thread |
| **Buffing** | Partial removal of the hose cover (to a specific diameter) to permit coupling insertion; uses a grinding wheel |
| **Burst Pressure** | Pressure that ruptures a hose; for testing purposes only — DO NOT use hose above published working pressure |
| **C (Cutoff Length)** | The portion of a coupling NOT inside the hose; used to calculate cut hose length |
| **Caliper** | Tool to measure ID and OD within a thousandth of an inch |
| **Chemical Resistance** | Index rating how well a hose compound endures contact with chemicals (Gates ratings: 1 = continuous, 2 = temporary, NR = not recommended) |
| **Crimp** | Method of permanently attaching hose couplings by compressing the ferrule |
| **Crimp OD** | Outside diameter of the ferrule after crimping; measured with calipers at the ridges |
| **Dash Size** | Industry standard for hose size — ID in 1/16" increments (e.g., -8 = ½") |
| **DIN** | Deutsch Industrie Norme — German standards for couplings and fittings |
| **DOT/FMVSS** | Department of Transportation Federal Motor Vehicle Safety Standards for hydraulic, air, and vacuum brake hose |
| **Dynamic Pressure** | Pressure that fluctuates as the hydraulic system works |
| **Effusion** | Seepage of chemical molecules through the hose wall |
| **Electrostatic Discharge (ESD)** | Release of static electrical energy that can build up on insulated objects; can damage sensitive parts or create pinholes in PTFE hose |
| **EnviroFluid™** | Gates hose line with nitrile tube for biodegradable fluids up to 250°F |
| **Ferrule** | The outer shell of a two-piece coupling that fits over the hose end and is crimped to hold the assembly together |
| **Field Attachable Couplings** | Reusable couplings installed without crimping equipment; DO NOT re-use these components |
| **Fitting** | A connector or closure for fluid power lines and ports |
| **Fluid Compatibility** | The hose assembly (tube, cover, reinforcement, and couplings including O-rings) must all be chemically compatible with the fluid |
| **Flow Rate (GPM)** | Amount of fluid measured in gallons per minute flowing through a hose |
| **Friction (Fluid)** | Rubbing of fluid against hose walls; primary cause of pressure drop and heat generation |
| **Galvanic Action** | Chemical reaction producing electrical current when two dissimilar metals are used together; weakens connections |
| **GAZ** | French coupling standard using 24° flare with metric fine threads |
| **Hose Size** | The ID (nominal inside diameter); expressed in dash numbers (sixteenths of an inch) |
| **Hydraulic Cylinder** | Component that performs work; an outer shell with an inner metal tube that extends and retracts |
| **Hydrodynamic** | Hydro + Dynamic = liquids actively in motion |
| **Hydrostatic** | Hydro + Static = liquids having no motion; force multiplication through static confined liquids |
| **ID** | Inside Diameter |
| **Impulse** | Pressure surge usually caused by the hydraulic pump |
| **JIC 37° Flare** | Joint Industry Conference; 37° flare seat; most common high-pressure fitting in North America |
| **MegaCrimp®** | One-piece permanent coupling with ferrule attached to stem |
| **MegaSys®** | Gates hose family with ½ SAE bend radius |
| **Minimum Bend Radius** | Tightest bend a hose can make and still pass SAE impulse testing at minimum bend radius |
| **MSHA** | Mine Safety and Health Administration — specifies flame resistance for underground mining hose |
| **NPTF** | National Pipe Taper Fuels — tapered threads that seal by deforming on the threads |
| **OD** | Outside Diameter |
| **O-Ring Boss** | Straight-thread fitting that seals with an O-ring compressed against the face of the port |
| **Operating Pressure (PSI)** | The hose must be capable of continuously handling system pressure including surges and spikes |
| **Permeation** | Seepage of fluid through the hose wall |
| **Power Crimp®** | Two-piece permanent coupling (separate stem and ferrule) |
| **Pressure Drop** | Difference in pressure between fluid entering and exiting a hose assembly; caused by friction, viscosity, length, size, and fittings |
| **Pressure Spike** | Sudden dramatic pressure increase (e.g., backhoe hitting a rock) |
| **PTFE** | Polytetrafluoroethylene — excellent high-temperature and chemical-resistance properties |
| **Reinforcement** | The structural layer of a hose (between tube and cover) that provides pressure resistance |
| **Reservoir** | Stores fluid; keeps oil clean; controls temperature; removes air |
| **SAE** | Society of Automotive Engineers — establishes American standards for hydraulic hose |
| **Skive** | Removing the rubber cover of a hose to expose the reinforcement where the fitting attaches |
| **Sleeving** | Protective covering for hoses; most common is nylon |
| **Spike** | See Pressure Spike |
| **STAMPED** | Acronym for hose selection: **S**ize, **T**emperature, **A**pplication, **M**aterial, **P**ressure, **E**nds of couplings, **D**elivery |
| **Static Pressure** | Steady-state pressure in a confined system |
| **Stem** | The inner part of a coupling inserted into the hose |
| **Swage** | Similar to crimping but the coupling is forced into stationary dies |
| **TuffCoat™** | Gates corrosion-resistant coupling plating — 400+ hours rust protection |
| **Tube** | Innermost hose layer; carries the fluid; determines hose size (ID) |
| **USCG** | United States Coast Guard — specifies requirements for marine hose (SAE J1475 and J1942) |
| **Viscosity** | Measure of a fluid's ability to flow; higher viscosity = thicker fluid = greater pressure drop |
| **Working Pressure** | Maximum continuous operating pressure; hose must meet or exceed the system pressure including surges |
