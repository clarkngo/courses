## What Are Couplings?

Couplings are the **metal components** of a hydraulic hose assembly. The **stem end** (threaded end) connects the assembly to other components in the hydraulic system.

There are **two types** of hydraulic couplings:
1. **Permanent** — require crimping or swaging equipment
2. **Reusable (Field-Attachable)** — can be installed with a vise and wrenches

> ⚠️ **Always use NEW components.** No permanent coupling part — hose, stem, or ferrule — is reusable.

---

## Permanent Couplings

### MegaCrimp® One-Piece Couplings
The ferrule is permanently attached to the stem. Designed for easy insertion — no lubrication required.

### Power Crimp® Two-Piece Couplings
- **Stem** — inserted into the hose
- **Ferrule** — the outside shell that fits over the hose end
- **NOTE:** Stem and ferrule must be compatible.

### Ferrule Types

| Type | Description | Trend |
|------|-------------|-------|
| **Skive ferrules** | Rounded serrations (teeth) | Becoming obsolete; requires extra time |
| **Non-skive ferrules** | Sharp serrations bite through the cover | Most popular in replacement market |

---

## Field-Attachable (Reusable) Couplings

Used where crimping equipment is unavailable. Can be installed with a vise and crescent wrenches. Types include:
- Skive
- No-skive
- Lock-on
- 100R5 (C5) DOT approved

> The coupling and the hose it fits on **must be compatible**.

## Coupling Identification

![GlobalSpiral and MegaCrimp coupling styles showing straight, 45°, 90°, and tee configurations](/courses/images/hydraulic-hose-fittings/m4/coupling-gallery.png)

To identify a coupling:
1. **Measure the ID or OD**
2. **Measure the threads per inch** (USA) or per millimeter (metric)
3. **Measure the angle of the mating seat**
4. **Compare all measurements** to the fitting dimensions in the Gates Hydraulics Catalog

## Gates Coupling Part Numbering System

Every Gates coupling follows a standardized part number that tells you exactly what you have.

### Example: **12 GS - 12 FJX 90L**

![Gates coupling part number diagram: 12 GS - 12 FJX 90L with annotations: 12 = Hose Dash Size (3/4"), GS = Stem Type (GlobalSpiral), 12 = Stem Size, F = Female, J = JIC, X = Swivel, 90 = Degree of Bend, L = Drop Length Long](/courses/images/hydraulic-hose-fittings/m4/coupling-part-number.png)

| Code | Meaning |
|------|---------|
| **12** (first) | Hose Dash Size (3/4") |
| **GS** | Stem Type — GlobalSpiral™ |
| **12** (second) | Stem Size |
| **F** | Female |
| **J** | JIC (37° Flare) |
| **X** | Swivel |
| **90** | 90° Degree of Bend |
| **L** | Drop Length (Long) |

### Gates Stem Style Codes
- **G20** = GlobalSpiral™
- **G25** = MegaCrimp®
- **G40** = C14 (PTFE)
- **G45** = PolarSeal®
- **G9** = Quick Disconnects

### Global Part Number — Example: **G25100-0808**
- **G25** = MegaCrimp® series
- **100** = Thread configuration (Male Pipe)
- **08** = Stem size (-8 = ½")
- **08** = Thread size (-8 = ½")

## Thread Measurement

![Thread measurement diagrams showing how to read male OD and female ID dimensions, and use of a pitch gauge for thread count identification](/courses/images/hydraulic-hose-fittings/m4/thread-measurement.png)

**Key rule:** When measuring threads with a pitch gauge, keep the **center lines parallel** — not at an angle — to get an accurate reading.

---

## North American Thread Standards

There are **9 North American standard thread types**:

| Thread Type | Key Feature | Notes |
|-------------|-------------|-------|
| **National Pipe Thread (NPTF/NPSF/NPSM)** | Tapered threads; seal on threads | NPTF male mates with NPTF, NPSF, and NPSM female |
| **JIC 37° Flare (MJ/FJX)** | Straight threads, 37° flare seat | Most popular; seal made on flared seat |
| **SAE 45° Flare (MS/FSX)** | Straight threads, 45° flare seat | Gates marks hex with a single notch to distinguish from JIC |
| **O-Ring Boss (MB/FB/MBX)** | Straight threads; O-Ring seals on port face | Very reliable; resists vibration |
| **Flat Face O-Ring (FFOR/MFFORX)** | O-ring in groove on flat male face | Seal when O-ring contacts flat female face |
| **SAE Inverted 45° Flare (MIX)** | Straight threads; 45° inverted flare | Male mates only with inverted flare female |
| **SAE Code 61 Flange (FL)** | 4-bolt split flange | High-pressure; used on pumps and motors worldwide |
| **SAE Code 62 Flange (FLH)** | Thicker, higher pressure than Code 61 | Extremely high-pressure lines |
| **Flareless Tube / MFA** | Compression sleeve + 24° seat | Connects hose to metal tubing |

> **37° vs 45° Tip:** The JIC 37° male/female have **no markings** on the hex. The SAE 45° male/female have **one notch** on each hex. Measure the seat angle to confirm.

> **24° Cone** — Also used on many high-pressure applications: metric standpipe, metric bite sleeve, and German DIN / French GAZ fittings all use the 24° compression termination.

---

## British Thread Standards

British couplings are widely used in the UK, Europe, Japan, and Commonwealth countries (India, Australia, New Zealand) and are becoming increasingly common in North America.

| Type | Gates Code | Key Feature |
|------|-----------|-------------|
| **BSPP** | MBSPP | Straight threads, 30° seat; seal with O-ring or soft metal washer |
| **BSPT** | MBSPT | Tapered threads; seal on threads; identical to Japanese tapered pipe thread |

> BSPP and NPSM look similar but are **NOT interchangeable** — different thread pitch and 55° vs 60° angle.

---

## Japanese Thread Standards

| Type | Gates Code | Key Feature |
|------|-----------|-------------|
| **JIS 30° Flare** | MJIS/FJISX | Straight threads, 30° seat — looks like BSPP but seat is funnel-shaped (not inverted) |
| **Japanese Tapered Pipe** | MBSPT/FBSPT | **Identical and fully interchangeable** with British BSPT |
| **Komatsu 30° Flare** | MK/FKX | Metric fine threads, 30° seat; 2 notches on hex to distinguish from JIS |
| **Komatsu Flange** | FLK | Interchangeable with SAE Code 61; use SAE O-ring when replacing |

> 🔑 Japanese 30° flare and BSPP threads are the same, but their seats are different — **not interchangeable**.

---

## German DIN Standards

German DIN couplings are used primarily in Germany and parts of Europe.

| Type | Key Feature |
|------|-------------|
| **DIN 24°** | Most common; 24° seat; mates with female 24° O-Ring or female universal 24°/60° cone |
| **DIN 60°** | Less common; 60° cone seat; mates with female universal 24°/60° cone |
| **DIN 3852 Type A** | Straight threads; O-Ring seal on face of port |
| **DIN 3852 Type B** | Straight threads; face seal |
| **DIN 3852 Type C** | Tapered threads; seal on threads; three series: extra light (LL), light (L), heavy (S) |

> **DIN Light vs Heavy:** Measure the tube ID — heavy series has thicker walls so the ID is smaller.

---

## French (GAZ) Standards

| Type | Key Feature |
|------|-------------|
| **GAZ 24° Flare** | 24° seat, metric fine threads; mates with 24° female cone or female tube coupling |
| **GAZ Poclain 24° Flange** | High-pressure specialty fitting; used on Poclain equipment; seal on 24° seat |

---

## Additional Coupling Selection Criteria

When selecting a coupling for any application, also consider:

| Factor | Key Rule |
|--------|----------|
| **Corrosion resistance** | Gates uses TuffCoat™ — 400+ hours rust protection (500% better than standard 72-hr salt spray) |
| **Vibration** | Use O-Ring or split-flange couplings — avoid thread-seal couplings |
| **Temperature** | Use O-Ring couplings; never use brass or aluminum in high-temp applications |
| **Fluid compatibility** | Check chemical resistance charts for both coupling material AND O-Rings |
| **Pressure** | Don't use swivel staked nut couplings at extremely high pressures; no aluminum ferrules on high-impulse |
| **Galvanic action** | Avoid mixing dissimilar metals — chemical reaction creates electrical current that weakens connections |
